### Installation In Local Environment
- Create .env file after editing this example
```
NODE_ENV = "production"
PORT = "5000"
MONGO_URI = "mongodb+srv://username:pass@cluster.mongodb.net/"
JWT_SECRET = "abc123"
```

- Building & running the application 

```
npm install
cd everestminds && npm install && npm build
cd ..  && npm start
```
